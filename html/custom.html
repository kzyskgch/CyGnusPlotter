——Cytosine-Guanine dinucleotide sites Plotter——

Custom mode: You can get a diagram from a sequence.

Link: simple mode

Title  

  Display the title 

This option enables the title to be displayed. Users can change the location of the displayed title.

   left center right

Sequence
 

Sites matching pattern

Enter [site name], [pattern], [position on the pattern] to each row.

The specified pattern will be automatically recognized and indicated.

Users can use the regular expression in Python.

example1) CpG,CG,0

example2) EcoRI,GAATTC,1

example3) BglI,GCC.{5}GGC,7

Boxes

Enter [box name], [start position], [end positon] to each row.

Assumed purpose of this component class is indicating exons.

Arrows

Enter [arrow name], [start position], [end positon] to each row.

Arrow is similar to Box. However, arrows indicate left or right depending on the input.

If positions inputted larger to smaller, it indicates rightward, and if inputted smaller to larger it indicates leftward.

Probes

Enter [Probe name], [start position], [end positon] to each row.

Specified ranges are displayed as horizontal lines

Clip Sequence

Checking this box and specifying the range enable to use clipped sequence range.

Display Shafts

Checking this box enables to display shafts indicating horizontal lines traversing the sequence.

 Display a scale Bar 

Checking this box enables of display scale bar. Users can change unit, length, and place of display.

Length:  Unit: 

left center right

 Auto Rescale 

This option enables adjustment of diagram widths as points.
